git branch
==========
El comando branch te permite crear ramas de tu codigó y verlas
```
git branch -v
```
el parámetro -v te muestra los branch locales
```
git branch -D <mi_branch>
```
El parámetro -D borra el branch en mención

```
git branch -r
```
El parámetro -r te muestra los branchs remotos
